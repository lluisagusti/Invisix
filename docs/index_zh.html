🚀 快速开始（10分钟）

1

刷写镜像 💾

  1. 从发布页面下载 invisix-distro.img。
  2. 下载 Raspberry Pi Imager。
  3. 选择：
    • 设备：Raspberry Pi 4
    • 操作系统："Use Custom" → invisix-distro.img
    • 存储：你的SD卡
  4. 重要：Next → "NO"（已配置好！）→ Write。
2

激活VPN（可选） ✨

如果只想在家拦截广告，跳过此步。

  1. 在 tailscale.com 创建免费账户。
  2. Settings → Keys → "Generate auth key"。
  3. 复制密钥 (tskey-...)。
  4. 打开SD卡 (bootfs)。
  5. 创建 tailscale-auth.key，粘贴密钥，保存。
  6. 弹出SD卡。
3

即插即用 🔌

  1. SD卡 → 树莓派。
  2. 网线 → 路由器连接到树莓派。
  3. 电源线 → 树莓派。
  4. 等待5分钟 ⏳ — 自动完成所有配置。
4

配置路由器 ⚙️

  1. 打开路由器管理页面 (http://192.168.1.1)。
  2. 找到 DHCP / 局域网 设置。
  3. 找到 raspberrypi → 启用静态地址分配。
  4. 复制其IP（例如：192.168.1.50）。
  5. 将该IP设为首选DNS服务器。

✅ 完成！
