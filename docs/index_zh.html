🚀 快速开始配置指南 (十分钟)

1

刻录镜像到存储卡 💾

  1. 请在 SourceForge 上下载最新发布的 **`invisix.img`** 包文件。
  2. 下好并配置使用 树莓派的烧录工具(Raspberry Pi Imager)。
  3. 开启这个烧录神器：
    • 选择设备模型: 找到对应的 Raspberry Pi 4
    • 挑选OS类型: 滚动列表底端 → "Use Custom(自定义源)" -> 选中前面放好的 invisix.img
    • 存储载体器位置: 指定读卡器上刚插入的空白 SD 卡
  4. 最关键步骤: 不要点击"编辑参数设定"(如果你被提示选框拦截，勇敢按下**NO(也就是不启用改数)**)。
  5. 放心痛快的点击右下角 Write (开始写入)。然后就歇一会儿等待它最终顺利走完流程。
2

激活获取无偿 VPN 连接资格 (可选项任务) ✨

如果您不使用异地移动网络组建的加密联线服务请放心忽略此项，只为过滤本地端流氓广告不用做这个步骤。

  1. 在 tailscale.com 完成最基础的信息账户创立。
  2. 在其后台仪表盘上点击左侧 Settings(配置) > Keys(密钥管理) 并找到此按钮操作点进去： "Generate auth key (新增密钥)"。
  3. 把生成的密钥完全无误的复制出来。(外观通常像 tskey-...)。
  4. 回桌面，将装有刚刻得热腾腾的树莓派系统 SD 盘读卡器插上电脑 (系统通常会自动挂载弹出叫作 bootfs 的磁盘卷分区)。
  5. 在它的根目录下建立一个新的文本记事簿并赋予名字： tailscale-auth.key, 接着把那串拷贝到的私家密文放置其中。
  6. 搞定一切并存好盘、点出图标拔退这块神圣载玻片。
3

给硬件通电！🔌

  1. 把刚才做好的 SD 芯片推到板卡的小金属槽中卡紧卡稳。
  2. 抽出一根物理实心以太铜网线连接好主网关(或软路由交换机)。
  3. 插座那一边给予可靠平稳的电流传输。
  4. 这时候不要去碰它，给系统自我释放和初始化布署大概 5分钟时间⏳ — 程序自己能够将后续最枯燥庞大的工程自动化打理完毕。
4

与路由器对话交接授权 (就这一个环节必须要人来手控) ⚙️

  1. 前往你平日拨号拨打宽带时那个光猫猫或是家里大号路由的总入口后台网页管理界面(通常它的默认门牌是 http://192.168.1.1 也有些叫 http://192.168.0.1 ).
  2. 进去了在各个抽屉（分页栏卡）里面细心寻宝找名叫 DHCP或者本地内部局区域网参数 的地方。
  3. 搜索到名单成员列表中出现了那位叫做 `raspberrypi` 名字的客户人员，勾选它旁边的设定成 "Static Lease （静态死锁分配 / 固化常态化IP颁布绑定)" (如此一来断电再上也是这个专用座位码不会被别人占用来惹乱局子了)。
  4. 把这串数值保留好复制住。(例：长得像这样 192.168.1.50 的内网号段)。
  5. 最后去配置面板的首个主控字段寻找**Primary DNS(网络默认解析首当其冲请求第一选择者)**，粘贴换作并确定就是刚才那位同志的那个数字号牌。保存使其生效重启。
