üöÄ Quickstart (10 minutes)

1

Flash the Image üíæ

  1. Download the invisix.img from SourceForge.
  2. Download and install Raspberry Pi Imager.
  3. Open the program and select:
    • Device: Raspberry Pi 4
    • OS: "Use Custom" -> Select invisix-distro.img
    • Storage: Your SD Card
  4. IMPORTANT: Click NEXT and then "NO" to edit settings (already configured!).
  5. Click Write and wait.
2

Activate VPN (Optional) ‚ú®

Skip if you only want ad blocking at home.

  1. Create a free account at tailscale.com (Google/Microsoft login works).
  2. Go to Settings > Keys and click "Generate auth key".
  3. Copy the key (tskey-...).
  4. Open the flashed SD card on your computer (disk named bootfs).
  5. Create a file named tailscale-auth.key and paste the key inside. Save.
  6. Eject the SD card.
3

Plug & Play üîå

  1. SD card ‚Üí Raspberry Pi.
  2. Ethernet cable ‚Üí Router to Pi.
  3. Power cable ‚Üí Pi.
  4. Wait 5 minutes ‚è≥ ‚Äî it auto-configures everything.
4

Tell Your Router (The one low-technical configuration change) ‚öôÔ∏è

  1. Open your router's page (http://192.168.1.1 or http://192.168.0.1).
  2. Find DHCP or Local Network settings.
  3. Find raspberrypi ‚Üí enable "Static Lease" (so its IP never changes).
  4. Copy its IP (e.g., 192.168.1.50).
  5. Set that IP as your network's Primary DNS server.
