Download and Flash ğŸ’¾

  1. Download the invisix-distro.img file.
  2. Download and install Raspberry Pi Imager.
  3. Open the program and select:
    • Device: Raspberry Pi 4
    • OS: "Use Custom" -> Select invisix-distro.img
    • Storage: Your SD Card
  4. IMPORTANT: Click NEXT and then "NO" to edit settings (we pre-configured them).
  5. Click Write and wait.
2

Configure VPN (Magic!) âœ¨

Optional: skip if you don't want VPN access.

  1. Go to Tailscale.com and create a free account.
  2. Go to Settings > Keys and click "Generate auth key".
  3. Copy the key starting with tskey-....
  4. On your computer, open the flashed SD card (disk named bootfs).
  5. Create a text file named tailscale-auth.key.
  6. Paste your key inside, save, and close.
  7. Eject the SD card.
3

Connect and Power On ğŸ”Œ

  1. Insert SD card into Raspberry Pi.
  2. Connect Ethernet cable (Router <-> Pi).
  3. Connect power.
  4. Wait 5 minutes â³. The Pi will auto-configure itself.
4

Configure Router âš™ï¸

The only manual step! Tell your router to use the Pi for DNS.

  1. Log in to your router (e.g., http://192.168.1.1).
  2. Find DHCP or Local Network settings.
  3. Find the device named raspberrypi and enable "Static Lease" or "DHCP Reservation".
  4. Copy its IP address (e.g., 192.168.1.50).
  5. In DNS settings, set that IP as the Primary DNS.
