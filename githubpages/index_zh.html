📦 您需要什么？

  • Raspberry Pi 4 (或更高版本)。
  • MicroSD 卡 (至少 8GB)。
  • 以太网线。
  • Pi 的 USB-C 电源。

🚀 安装指南

1

下载并烧录 💾

  1. 下载 invisix-distro.img 文件。
  2. 下载并安装 Raspberry Pi Imager。
  3. 打开程序并选择：
    • 设备： Raspberry Pi 4
    • 操作系统： "Use Custom" -> 选择 invisix-distro.img
    • 存储： 您的 SD 卡
  4. 重要提示： 点击 NEXT，然后对编辑设置选"NO"（我们已预先配置好）。
  5. 点击 Write 并等待。
2

配置 VPN (魔法！) ✨

可选：如果不需要 VPN 访问，请跳过。

  1. 前往 Tailscale.com 创建免费账户。
  2. 前往 Settings > Keys 并点击 "Generate auth key"。
  3. 复制以 tskey-... 开头的密钥。
  4. 在电脑上打开烧录好的 SD 卡（名为 bootfs 的磁盘）。
  5. 创建一个名为 tailscale-auth.key 的文本文件。
  6. 将密钥粘贴进去，保存并关闭。
  7. 弹出 SD 卡。
3

连接并通电 🔌

  1. 将 SD 卡插入 Raspberry Pi。
  2. 连接以太网线（路由器 <-> Pi）。
  3. 连接电源。
  4. 等待 5 分钟 ⏳。Pi 将自动配置。
4

配置路由器 ⚙️

唯一的从手动步骤！告诉路由器使用 Pi作为 DNS。

  1. 登录路由器（例如 http://192.168.1.1）。
  2. 找到 DHCP 或 局域网 设置。
  3. 找到名为 raspberrypi 的设备并启用 "Static Lease" 或 "DHCP Reservation"。
  4. 复制其 IP 地址（例如 192.168.1.50）。
  5. 在 DNS 设置中，将该 IP 设置为 首选 DNS。

✅ 准备就绪！
